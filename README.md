# LinkTree

WebサイトのURLを入力すると、サイトのリンク構造をツリー状に表示するアプリです。

スクレイピングの技術を使用しております。

<br><br>

以下のリンクにアクセスすると、Webアプリ「MeasRec」のページに移動します。

・[LinkTree](http://44.212.209.108/)
